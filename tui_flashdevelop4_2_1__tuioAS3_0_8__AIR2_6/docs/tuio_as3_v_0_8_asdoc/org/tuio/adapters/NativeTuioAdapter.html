Classpublic class NativeTuioAdapter
InheritanceNativeTuioAdapter AbstractTuioAdapter Object

Listens on native TouchEvents. All touch devices supported by Adobe AIR (e.g., Windows 7 touch and Android touch devices) can be used with NativeTuioAdapter. Native touch events will be translated to TuioTouchEvents. Hence, applications created with TUIO AS3 can be used with native touch hardware by employing NativeTuioAdapter.
