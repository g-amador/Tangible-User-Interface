Packageorg.tuio.adapters
Classpublic class AbstractTuioAdapter
InheritanceAbstractTuioAdapter Object
Subclasses MouseTuioAdapter, NativeTuioAdapter, TuioClient

Provides basic functionality for a Tuio adapter. This can either be a real Tuio adapter like TuioClient or an adapter that simulates Tuio functionality like MouseTuioAdapter or NativeTuioAdapter.

getTuioBlob()method 
public function getTuioBlob(sessionID:Number, source:String = null):TuioBlob

Retrieves the TuioBlob fitting the given sessionID and source.

Parameters

sessionID:Number — The sessionID of the designated tuioBlob
 
source:String (default = null) — The source message of the TUIO message provider. If null, all TuioBlob source messages will be searched for the TuioBlob with the appropriate sessionID. Attention: If there are more than one TuioBlob with sessionID the first appropriate TuioBlob will be returned.

Returns
TuioBlob — The TuioBlob matching the given sessionID. Returns null if the tuioBlob doesn't exists
