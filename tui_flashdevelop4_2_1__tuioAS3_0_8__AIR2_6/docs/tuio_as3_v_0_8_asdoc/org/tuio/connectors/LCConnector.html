Classpublic class LCConnector
InheritanceLCConnector Object
Implements IOSCConnector

This implementation of the IOSCConnector uses Flash's LocalConnection to receive and send OSC data. It is primarily meant to be used with Georg Kaindl's UPD -> LC bridge: http://gkaindl.com/software/udp-flashlc-bridge Though you can run into problems depending on the tracker and the amount of data that is transmitted. If you want to transmit OSC bundles between two Flash instances this connector probably will do better than UDP. Note that the default values for the connection names used by the constructor do not support this out of the box.
