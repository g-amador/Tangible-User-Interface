Classpublic class TuioLegacyListener
InheritanceTuioLegacyListener flash.events.EventDispatcher
Implements ITuioListener

Adopts function of TUIO class from Touchlib's Tuio AS3 framework. All functions of TUIO can be found within TuioLegacyListener. Additionally, TuioLegacyListener dispatches TouchEvents Touchlib's Tuio AS3 framework. Thus, existing multitouch software that has been built with Touchlib's Tuio AS3 framework can be easily ported to the TUIO AS3 framework. All existing calls to TUIO must be replaced with TuioLegacyListener.getInstance() and the import paths of TouchEvent have to be changed to org.tuio.legacy.TouchEvent. Following legacy functions have been transfered from TUIO to TuioLegacyListener: - listenForObject - getObjectById TuioLegacyListener implements the interface ITuioListener. Thus, it is being called whenever a Tuio object, cursor or blob is being received. When a cursor (== a touch) is being received in the function addTuioCursor, addTuioCursor looks if there is a DisplayObject under the cursor and dispatches a TouchEvent.MOUSE_DOWN on this DisplayObject. Additionally, a TouchEvent.MOUSE_DOWN is being dispatched on the stage. Use only for the port of existing code to TUIO AS3. For the current Tuio event implementation see:

public function addTuioCursor(tuioCursor:TuioCursor):void

dispatches TouchEvent.MOUSE_DOWN and TouchEvent.MOUSE_OVER events on the DisplayObject under the touch. Additionally, the same events are being dispatched on the stage object in order to provide the possibility to objects to listen on TouchEvents that are being dispatched on the stage (useful for the implementation of dragging and so on).

getObjectById()method 
public function getObjectById(id:uint):TUIOObject

takes over the duties of the function getObjectById from TUIO. getObjectById looks in the global cursor list for the tuio element with the id id. If this element is being found it will be repackaged into a (legacy) TUIOObject instance and returned to the caller.

takes over the duties of the function getObjects from TUIO. getObjects puts all elements from the global cursor list (tuioClient.getTuioCursors()) into TUIOObjects and adds them to an array that is being returned.
