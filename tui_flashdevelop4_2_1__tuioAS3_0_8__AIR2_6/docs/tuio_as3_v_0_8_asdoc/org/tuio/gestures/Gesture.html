Classpublic class Gesture
InheritanceGesture flash.events.EventDispatcher
Subclasses OneDownOneMoveGesture, OneFingerMoveGesture, PressTapGesture, ThreeFingerMoveGesture, TwoFingerMoveGesture

This class is thought as a base class for creating custom gestures for the GestureManager. To create a custom gesture you have to extend this class and overwrite dispatchGestureEvent function.

Gestures are described by a sequence of steps. If all steps are saturated the dispatchGestureEvent is called. For more information on gesture steps have a look at the GestureStep asdoc.

To track the progress in the GestureStepSequences you can add an event listener for GestureStepEvent to your extension of Gesture
