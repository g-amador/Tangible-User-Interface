Packageorg.tuio
Interfacepublic interface ITuioListener
Implementors FiducialTuioAS3LegacyListener, TuioDebug, TuioLegacyListener, TuioManager

This interface defines callback functions that will be called by the TuioClient if the implementing class was added as a listener to the according TuioClient.
