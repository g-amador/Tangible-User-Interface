Packageorg.tuio.fiducial
Classpublic class TuioFiducialDispatcher
InheritanceTuioFiducialDispatcher flash.events.EventDispatcher
Implements ITuioListener

The TuioFiducialDispatcher class offers an easy to use global listener concept for the use of fiducials in an application.

Receivers can register themselves with TuioFiducialDispatcher in order to receive callbacks for a certain fiducial id. A receiver must implement ITuioFiducialReceiver.

invertRotationproperty
invertRotation:Boolean

Some trackers invert the rotation of a fiducial. Thus, by setting invertRotation true the rotation of a fiducial will be inverted.


Implementation

rotationShiftproperty 
rotationShift:Number

Fixed degree value, which is added to the rotation value. Trackers behave differently in how they calculate the rotation of a fiducial on their surface. Thus, rotationShift can be set according to a tracker's properties. The simulator does not need any shift.
