Packageorg.tuio.fiducial
Classpublic class FiducialEvent
InheritanceFiducialEvent flash.events.Event

FiducialEvent is a high-level implementation of TuioEvent that offers only information that is connected to TuioObjects. It is the fiducial analogon to TouchEvent.
