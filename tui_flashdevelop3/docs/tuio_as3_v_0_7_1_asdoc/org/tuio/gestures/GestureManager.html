Classpublic class GestureManager
InheritanceGestureManager flash.events.EventDispatcher

addToIgnoreList()method 
public function addToIgnoreList(item:DisplayObject):void

Adds the given DisplayObject to an internal list of DisplayObjects that won't receive TouchEvents. If a DisplayobjectContainer is added to the list its children can still receive TouchEvents. The touchTargetDiscoveryMode is automatically set to TOUCH_TARGET_DISCOVERY_IGNORELIST.

Parameters

item:DisplayObject — The DisplayObject that should be ignored by TouchEvents.

init()method 
public static function init(stage:Stage, tuioClient:TuioClient):GestureManager

Intializes the GesturManager with the given TuioClient. If you have already initialized a TuioManager the given stage and tuioClient will be discarded and the ones specified for the TuioManager will be used instead. This behaviour is a result of the GestureManager using the TuioManager's events

Constant Detail
TOUCH_TARGET_DISCOVERY_IGNORELISTConstant
public static const TOUCH_TARGET_DISCOVERY_IGNORELIST:Number = 2

An ignore list is used to determin whether a TouchEvent is triggered on an InteractiveObject under the tracked point. Works for DisplayObject and subclasses.

TOUCH_TARGET_DISCOVERY_MOUSE_ENABLEDConstant 
public static const TOUCH_TARGET_DISCOVERY_MOUSE_ENABLED:Number = 1

The InteractiveObject's mouseEnabled parameter is used to determine whether a TouchEvent is triggered on an InteractiveObject under the tracked point. Works only for InteractiveObject and subclasses.

TOUCH_TARGET_DISCOVERY_NONEConstant 
public static const TOUCH_TARGET_DISCOVERY_NONE:Number = 0

The events will be triggered on the top object under the tracked point. Fastest method. Works for DisplayObject and subclasses.
